# Tema 11 — Indice

> **Titulo oficial**: Informatica basica. Representacion y comunicacion de la informacion: elementos constitutivos de un sistema de informacion. Caracteristicas y funciones. Arquitectura de ordenadores. Componentes internos de los equipos microinformaticos.
>
> **Bloque**: Parte II — Tecnico (Temas 11-40)
> **Nivel**: C1 — Tecnico Auxiliar TIC, Ayuntamiento de Madrid

---

## Estructura del tema

1. **Concepto de dato e informacion**
   1.1. Definicion de dato
   1.2. Definicion de informacion
   1.3. Clasificacion de datos (por funcion, naturaleza y variabilidad)
   1.4. Relacion dato-informacion-conocimiento

2. **Sistemas de informacion**
   2.1. Definicion y concepto
   2.2. Componentes basicos (hardware, software, datos, procedimientos, usuarios, red, retroalimentacion)
   2.3. Caracteristicas de un sistema de informacion
   2.4. Elementos funcionales (base de datos, transacciones, informes, procesos, usuarios, procedimientos)
   2.5. Funciones principales (entrada, almacenamiento, procesamiento, salida)
   2.6. Tipos de sistemas de informacion (DSS, TPS, EIS, MIS)
   2.7. Herramientas complementarias (Cuadro de Mando Integral, Datawarehouse)

3. **Arquitectura de ordenadores**
   3.1. Concepto de arquitectura de ordenadores
   3.2. Arquitectura Von Neumann (modelo, cuello de botella)
   3.3. Arquitectura Harvard (memoria separada, ventajas)
   3.4. Arquitectura Harvard modificada

4. **Generaciones de ordenadores**
   4.1. Primera generacion (1940-1952) — Valvulas de vacio
   4.2. Segunda generacion (1952-1964) — Transistores
   4.3. Tercera generacion (1964-1970) — Circuitos integrados (SSI/MSI)
   4.4. Cuarta generacion (1970-1981) — Microprocesador (LSI/VLSI)
   4.5. Quinta generacion (1981-1991) — IA, redes, interfaces graficas

5. **Componentes internos de los equipos microinformaticos**
   5.1. Buses (control, direcciones, datos)
   5.2. El procesador (CPU)
        5.2.1. Tecnologias: CISC vs RISC
        5.2.2. Medidas de potencia (FLOPS, MIPS)
        5.2.3. Multiprocesamiento y MultiThreading
   5.3. Unidad Central de Proceso
        5.3.1. Registros (datos, direcciones, estado, proposito general/especifico)
        5.3.2. Unidad de Control (reloj, contador de programa, decodificador, secuenciador)
        5.3.3. Unidad Aritmetico-Logica (ALU) — operaciones aritmeticas, logicas, desplazamiento
   5.4. Ciclo de ejecucion de instrucciones (fetch + ejecucion)
   5.5. Modos de direccionamiento (implicito, inmediato, directo, indirecto, relativo)

6. **Memoria**
   6.1. Clasificaciones (interna/externa, volatil/no volatil, jerarquia por velocidad)
   6.2. Memoria RAM
        6.2.1. Concepto y caracteristicas
        6.2.2. Evolucion (ferrita, DIP, SIPP, SIMM, DIMM, DDR/DDR2/DDR3/DDR4)
        6.2.3. Tipos especiales (RIMM, FB-DIMM, GDDR, SO-DIMM, ECC)
        6.2.4. Parametros (velocidad, ancho banda, Dual Channel, latencia CAS)
        6.2.5. RAM-CMOS
   6.3. Memoria ROM
        6.3.1. Concepto
        6.3.2. Tipos (ROM, PROM, EPROM, EEPROM, Flash)
   6.4. BIOS y UEFI (concepto, diferencias, Secure Boot)

7. **Medidas de capacidad de memoria**
   - Bit, Nibble, Byte, Palabra, KB, MB, GB, TB, PB, EB, ZB, YB

8. **Sistemas de numeracion**
   - Decimal (base 10), Binario (base 2), Octal (base 8), Hexadecimal (base 16)

9. **Juegos de caracteres**
   9.1. ASCII (7 bits, 128 caracteres)
   9.2. ASCII extendido (8 bits, paginas de codigos)
   9.3. Unicode y UTF-8 (longitud variable, compatibilidad, ventajas/desventajas)
   9.4. EBCDIC (sistemas mainframe IBM)

## Conceptos clave para memorizar

| Concepto | Dato clave |
|---|---|
| Bit | Unidad minima de informacion (0 o 1) |
| Byte | 8 bits |
| Von Neumann | Memoria unica para datos e instrucciones — cuello de botella |
| Harvard | Memorias separadas — permite acceso simultaneo |
| CISC | Instrucciones complejas, 4-10 ciclos (Intel x86, AMD) |
| RISC | Instrucciones simples, 1 ciclo (ARM, MIPS, SPARC) |
| DDR4 | 288 pines, 2133-4000 MHz, hasta 64 GB |
| ASCII | 7 bits, 128 caracteres, creado 1963 |
| UTF-8 | 1-4 bytes variable, compatible ASCII, estandar web |
| UEFI | Sucesor BIOS, Secure Boot, GPT, hasta 128 particiones |
| RAM | Volatil, acceso aleatorio, almacena datos en uso |
| ROM | No volatil, almacena firmware (BIOS/UEFI) |
| ALU | Operaciones aritmeticas y logicas (AND, OR, NOT, XOR) |
| Ciclo instruccion | Fetch (busqueda) + Ejecucion |
